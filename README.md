# 고려대학교 심리학부 주점 POS 시스템

단일 HTML 파일(`index.html`)로 동작하는 주점 카운터용 포스기 웹앱.  
별도 설치 없이 브라우저에서 바로 실행 가능.

---

## 화면 구성

| 영역 | 설명 |
|------|------|
| 상단 헤더 | 타이틀 + 엑셀 출력 버튼 |
| 통계 바 | 실시간 매출 현황 (항상 상단 고정) |
| 테이블 그리드 | 1~20번 테이블, 4열 × 5행 배치 |
| 주문 모달 | 테이블 클릭 시 팝업으로 주문 입력 |
| 확인 모달 | 취소·치움 등 중요 액션 전 확인 팝업 |

## 주요 기능

### 테이블 관리
- **빈 자리 / 영업 중** 상태를 아이콘과 색상으로 구분
- 첫 주문 저장 시 자동으로 **착석 시각** 기록
- 영업 중인 테이블 카드에 **현재 청구 금액** 표시

### 주문 입력
- 테이블 클릭 → 모달에서 메뉴별 수량 +/− 조절
- **할인 쿠폰** 수량 입력 (1장 = 1,000원 할인)
- 메뉴 합계 / 쿠폰 할인 / 청구 금액 실시간 자동 계산
- `주문 저장` 버튼으로 확정

### 시간 추적
- 착석 후 경과 시간을 **초 단위**로 실시간 갱신
- 테이블 카드와 주문 모달 양쪽에 동시 표시
- 형식: `MM:SS` / 1시간 이상 시 `H:MM:SS`

### 테이블 버튼
| 버튼 | 동작 |
|------|------|
| `주문 저장` | 현재 입력 내용을 테이블에 반영 |
| `주문 취소` | 해당 테이블의 주문 전량 초기화 (매출 미반영) |
| `자리 치움` | 주문 내역을 매출에 **누적 합산** 후 테이블 초기화 |

> 주문 취소와 자리 치움은 실수 방지를 위해 확인 팝업이 먼저 표시됨.

### 실시간 통계 바
| 항목 | 내용 |
|------|------|
| 총 매출 (쿠폰 후) | 자리 치움된 테이블 + 현재 영업 중 테이블 합산 |
| 쿠폰 할인 총액 | 사용된 쿠폰 금액 합계 |
| 메뉴 합계 (쿠폰 전) | 쿠폰 적용 전 순수 메뉴 금액 합계 |
| 영업 중 / 전체 | 현재 사용 중인 테이블 수 |

### 엑셀 출력
`📊 엑셀 출력` 버튼 클릭 시 `.xlsx` 파일 다운로드 (3개 시트 포함).

| 시트 | 내용 |
|------|------|
| 주문 내역 | 테이블별 착석·퇴석 시각, 착석 시간, 주문 내역, 청구 금액 |
| 매출 요약 | 총 메뉴 합계 / 쿠폰 할인 / 총 매출 |
| 메뉴별 집계 | 메뉴별 판매 수량 및 매출 소계 |

> 자리 치움 전 영업 중인 테이블도 `(영업 중)` 상태로 포함되어 출력됨.

---

## 사용 방법

1. `index.html` 파일을 브라우저로 열기
2. 테이블 클릭 → 메뉴 수량 입력 → `주문 저장`
3. 고객 퇴석 시 해당 테이블 클릭 → `자리 치움`
4. 영업 종료 후 `📊 엑셀 출력`으로 기록 저장
